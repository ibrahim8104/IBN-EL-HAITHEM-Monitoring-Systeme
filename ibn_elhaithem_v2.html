02 — TECHNICAL ARCHITECTURE

System Design

LAYER 01 // SENSING
Custom PCB + Sensors
A purpose-built electronic board integrating all environmental sensors. Designed for reliability, low power consumption, and compact deployment across multiple rooms simultaneously.
LAYER 02 // PROCESSING
ESP32-S3 Microcontroller
The ESP32-S3 serves as the brain of each node — handling sensor data acquisition, local processing, and wireless transmission via built-in Wi-Fi and Bluetooth connectivity.
LAYER 03 // GATEWAY
Raspberry Pi Gateway
A Raspberry Pi acts as the central hub — aggregating data from all sensor nodes, managing the network, and relaying information to the cloud dashboard in real time.
